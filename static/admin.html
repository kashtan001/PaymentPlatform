Geo requisites

Реквизиты выбранного GEO

GEO в ссылке всегда использует первый доступный комплект. Если удалить первый, после обновления страницы будет использоваться следующий.

Удалять можно любой комплект, включая последний. Пустые GEO остаются доступны в админке, но пропадают из генератора ссылок.

Для этого GEO пока нет реквизитов. После добавления первый комплект сразу станет рабочим для новых ссылок.
